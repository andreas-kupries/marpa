# Marpa DOCTOOLS

# Installation

   * Install the main marpa packages

   * Invoke `bash tools/make-parser` to generate the doctools parser
     packages from the SLIF grammar (See `g/`).

   * Use `build.tcl install` to compile and install the parser
     packages.
